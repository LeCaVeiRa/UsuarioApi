📌 API de Consulta de Usuários (.NET + JWT + MySQL)

API REST desenvolvida em .NET para consulta e gerenciamento de usuários, com autenticação baseada em JWT (JSON Web Token) e persistência de dados em MySQL.

🚀 Tecnologias Utilizadas
.NET (ASP.NET Core Web API)
Entity Framework Core
MySQL
JWT (JSON Web Token)
Swagger (documentação de endpoints)
